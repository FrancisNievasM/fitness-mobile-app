# fitness-mobile-app
Mobile app for tracking exercises, built with JavaScript and Capacitor.

Fitness Tracker Mobile
📱 Descripción

Aplicación mobile desarrollada como proyecto académico para el registro y seguimiento de actividades físicas. Permite gestionar ejercicios, visualizar información y navegar entre distintas pantallas dentro de una interfaz optimizada para dispositivos móviles.

🚀 Funcionalidades
Registro e inicio de sesión de usuarios
Registro de ejercicios realizados
Listado de actividades
Filtros por:
Última semana
Último mes
Histórico completo
Visualización de información resumida
Integración con mapa
🛠️ Tecnologías utilizadas
JavaScript (Vanilla)
HTML / CSS
Capacitor
📁 Estructura del proyecto
www/index.html → estructura principal de la app
www/js/code.js → lógica principal (eventos, navegación, funcionalidades)
▶️ Cómo ejecutar el proyecto

Clonar el repositorio:

git clone https://github.com/TU_USUARIO/fitness-tracker-mobile.git

Instalar dependencias:

npm install
Ejecutar en entorno web o compilar con Capacitor según configuración
